# React SPA Project

This is a Single Page Application (SPA) built using React.

## Features

* Fast and responsive UI
* Single Page Application (SPA)
* Component-based structure
* Smooth navigation without page reload

## Technologies Used

* React.js
* JavaScript
* HTML
* CSS

## How to Run

1. Install dependencies:
   npm install

2. Start the project:
   npm start
